# REST Assured — печатная шпаргалка

Комплект рассчитан ровно на 5 листов A4.

- `rest-assured-cheatsheet.pdf` — готовый файл для двусторонней или односторонней печати.
- `rest-assured-cheatsheet.html` — редактируемый исходник.

Рекомендуемые параметры печати PDF:

- бумага: A4;
- масштаб: 100% / Actual size;
- ориентация: книжная;
- поля: отсутствуют, они уже заложены в макет;
- двусторонняя печать: переворот по длинной стороне.

Страницы:

1. Начальный уровень — основа REST Assured.
2. Базовый+ — построение запросов и specifications.
3. Средний — ответы, extraction, DTO, JsonPath и schema.
4. Средний+ — auth, CRUD и cleanup.
5. Middle+ — архитектура, диагностика и checklist.
