# REST Assured с нуля — печатный учебный конспект

Это не короткая шпаргалка, а последовательный конспект на 10 листов A4 для человека,
который впервые работает с REST Assured. Материал идёт от устройства HTTP request/response
до DTO-first CRUD-сценария, независимости тестов и структуры рабочего проекта.

- `rest-assured-cheatsheet.pdf` — готовая печатная версия.
- `rest-assured-cheatsheet.html` — редактируемый исходник.

Печатать: A4, книжная ориентация, масштаб 100%, без дополнительных полей.
