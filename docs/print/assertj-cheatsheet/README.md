# AssertJ — печатная шпаргалка

Комплект рассчитан ровно на 5 листов A4 и дополняет задания урока 1.

- `assertj-cheatsheet.pdf` — готовый файл для печати.
- `assertj-cheatsheet.html` — редактируемый исходник.

Рекомендуемые параметры печати PDF:

- бумага: A4;
- масштаб: 100% / Actual size;
- ориентация: книжная;
- поля: отсутствуют — они уже заложены в макет;
- двусторонняя печать: переворот по длинной стороне.

Страницы:

1. Начальный уровень — fluent-цепочки, строки, числа и описания.
2. Базовый+ — коллекции, `Optional`, `Map` и выбор строгой проверки.
3. Средний — `extracting`, `tuple`, `filteredOn`, `flatExtracting`.
4. Средний+ — soft assertions, исключения, числа, даты и recursive comparison.
5. Middle+ — архитектура проверок, другие assertion-библиотеки и checklist.
