# JUnit 5 — печатная шпаргалка

Комплект рассчитан ровно на 5 листов A4 и дополняет задания урока 1.

- `junit-cheatsheet.pdf` — готовый файл для печати.
- `junit-cheatsheet.html` — редактируемый исходник.

Рекомендуемые параметры печати PDF:

- бумага: A4;
- масштаб: 100% / Actual size;
- ориентация: книжная;
- поля: отсутствуют — они уже заложены в макет;
- двусторонняя печать: переворот по длинной стороне.

Страницы:

1. Начальный уровень — устройство JUnit, структура теста и assertions.
2. Базовый+ — lifecycle и параметризованные тесты.
3. Средний — исключения, timeout и assumptions.
4. Средний+ — Nested, dynamic tests, tags и extensions.
5. Middle+ — архитектура, диагностика и checklist теста.
